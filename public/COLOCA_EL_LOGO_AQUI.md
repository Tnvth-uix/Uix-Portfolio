# Logo

Guarda el logotipo **uiX** en esta carpeta con el nombre exacto:

```
public/logo.png
```

La app lo usa automáticamente en el header y el login (`components/atoms/Logo.tsx` → `/logo.png`).
Mientras el archivo no exista, se muestra un wordmark de respaldo.

Recomendado: PNG con fondo transparente, alto ≥ 200 px.

Puedes borrar este archivo una vez que hayas colocado `logo.png`.
